## 👨‍💻 About Me  

I am a highly motivated SDET and Full Stack Developer from India, passionate about building reliable and scalable software through a balanced approach to both development and quality engineering.

My professional journey began with automation testing using Selenium, TestNG, Rest Assured, Playwright, and Cucumber BDD, which helped me develop a strong understanding of application workflows, edge cases, and system stability.

Alongside testing, I actively design and develop web applications using the MERN Stack, and build backend services with Java and Spring Boot, enabling me to approach problems with both a tester’s mindset and a developer’s perspective.

I strongly believe that great software is achieved when quality, performance, and user experience work together. I continuously strive to learn, improve, and contribute with commitment and ownership.

🎯 My goal is to grow as a Software Engineer who can confidently design, test, and deliver high-quality software solutions.
